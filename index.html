User Experience Task

For the User experience, I was considering should I design the website as a single-scroll website or interactable website which has buttons to change to different contents, however I am heavily inspired by the apple and playstation website, therefore this whole website is designed as a single-scroll website. As my priority user is Professor Mary Jackson, an interactable website will definitely gain more marks in the point of view of myself, but when I looked at the apple and playstation website, especially in the pages with specific product, they have decided to use single-scroll website too. By using single-scroll design, it is more likely to be simple but including a much more modern design.

Comparing Single-Scroll website and Interactable website, it does rely on user preferences and the device user used to view the website. When the user is using smartphones to view a Single-Scroll website, it will be relatively easier because all they need to do is to keep scrolling the page and they do not require any further movement of their fingers, and this also suits user who uses PC or laptop to view the website. However viewing an Interactable website requires more movement of user's hands, and for some user like professor Stephen William Hawking, who has difficulty on moving their body or hands, it is definitely giving a terrible user experience for them.
